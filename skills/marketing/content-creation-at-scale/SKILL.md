# Content Creation At Scale

## Quick Start

This skill helps you build scalable content production systems. For complete frameworks, see the [Complete Guide](references/guide.md).

## Execution Checklist

### Building Content System

- [ ] Create content brief template
- [ ] Build style guide
- [ ] Recruit/train writers (in-house or freelance)
- [ ] Set up project management (Airtable, Asana)
- [ ] Define workflows (assign → draft → edit → publish)
- [ ] Create QA checklist
- [ ] Batch production (4-8 pieces at once)
- [ ] Track metrics per writer
- [ ] Optimize process monthly

## Key Components

- **Content Briefs**: Template with keyword, outline, examples, word count
- **Editorial Calendar**: 4-8 weeks ahead
- **Writer Network**: 3-5 reliable freelancers or AI + editor
- **QA Process**: Checklist before publish (SEO, brand, accuracy)

## Quick Wins

- Use AI for draft creation (Claude, GPT)
- Batch content briefs (write 10 at once)
- Standard operating procedures (SOPs) for each step
- Track time-to-publish, optimize bottlenecks
