# Lead Magnet Creation

## Quick Start

This skill helps you create high-value lead magnets that convert visitors into email subscribers. For complete strategies and examples, see the [Complete Guide](references/guide.md).

## Tools & Scripts

**Checklist Formatter** (`scripts/checklist_formatter.py`):

```bash
python scripts/checklist_formatter.py input.md output.pdf
```

Converts markdown checklists into professional PDF format.

## Execution Checklist

### Creating a Lead Magnet

- [ ] Identify specific pain point (one clear problem)
- [ ] Choose format (checklist, template, guide, toolkit, or swipe file)
- [ ] Create outline (5-10 actionable items)
- [ ] Write content (focus on immediate value)
- [ ] Design or format (professional,branded)
- [ ] Create landing page for download
- [ ] Set up email delivery automation
- [ ] Promote (ads, blog CTAs, social)
- [ ] Track conversion rate and optimize

## Key Frameworks

- **Lead Magnet Criteria**: Specific + Immediate + Actionable + Valuable
- **Format Selection**: Quick win = Checklist; Deep value = Guide; Reusable = Template
- **Naming Formula**: `[Number] [Format] to [Achieve Result] in [Timeframe]`
- **Delivery**: Email immediately, add to nurture sequence

## Common Lead Magnet Types

- **Checklist**: Quick reference, easy to consume (10 items)
- **Template**: Copy-paste tool (email, spreadsheet, doc)
- **Guide**: Educational PDF (10-20 pages)
- **Toolkit**: Bundle of resources (multiple templates + checklist)
- **Swipe File**: Examples to steal from

## Metrics Benchmarks

- **Opt-in rate**: 10-25% (landing page visitors)
- **Email open rate**: 40-60% (delivery email)
- **Download rate**: 80%+ (of opt-ins)

## Quick Wins

- Make it specific ("Cold Email Templates" not "Email Guide")
- Deliver instant value (10 minutes or less to use)
- Professional design (use Canva templates)
- Clear next step (nurture sequence after download)
- Gate with name + email only (don't ask for more)
