# Content Repurposing

## Quick Start

This skill helps you transform one piece of content into 10+ formats. For complete frameworks, see the [Complete Guide](references/guide.md).

## Execution Checklist

### Repurposing Workflow

- [ ] Choose pillar content (long-form blog, webinar, podcast)
- [ ] Extract key insights (5-10 main points)
- [ ] Create LinkedIn posts (one per insight)
- [ ] Create Twitter thread (summarize key points)
- [ ] Design infographic (visualize data/framework)
- [ ] Record short video (2-3 min summary)
- [ ] Create email newsletter (link to original + highlights)
- [ ] Extract quotes for social media
- [ ] Build slide deck presentation
- [ ] Update / expand for pillar page

## Key Frameworks

- **Content Pyramid**: 1 pillar piece → 10 micro-pieces (blog → social posts, emails, graphics)
- **Platform-Specific Adaptation**: Optimize for each channel (LinkedIn articles, Twitter threads, Instagram carousels)
- **Evergreen Refresh**: Update and republish top-performing content with new data
- **Atomization Strategy**: Break down complex content into standalone components

## Key Formats

- **1 Blog Post** → 10 LinkedIn posts + 5 Twitter threads + 1 infographic + 3 videos + 1 email + 1 slide deck
- **1 Webinar** → Blog post + video clips + podcast episode + social posts
- **1 Podcast** → Blog transcript + audiograms + quote graphics + email

## Quick Wins

- Use transcript tools (Otter.ai, Descript)
- Batch-repurpose weekly (Friday afternoons)
- Template social posts for quick editing
- Schedule across 30 days (drip content)
