# Social Campaign Orchestration

## Quick Start

This skill helps you run coordinated social media campaigns. For complete frameworks, see the [Complete Guide](references/guide.md).

## Execution Checklist

### Running Social Campaign

- [ ] Define campaign goal and KPIs
- [ ] Choose platforms (LinkedIn, Twitter, Instagram, Facebook)
- [ ] Create campaign hashtag
- [ ] Develop content calendar (20-30 posts)
- [ ] Create platform-specific assets
- [ ] Schedule posts across 2-4 weeks
- [ ] Engage with comments daily
- [ ] Track performance (reach, engagement, conversions)
- [ ] Adjust strategy mid-campaign
- [ ] Report results and learnings

## Key Frameworks

- ** Campaign Arc**: Launch (week 1) → Peak (week 2-3) → Wind down (week 4)
- **Content Mix**: 60% value, 30% engagement, 10% promotional
- **Posting Frequency**: LinkedIn (1x/day), Twitter (3-5x/day), Instagram (1-2x/day)
- **Engagement**: Respond within 2 hours for maximum algorithm boost

## Quick Wins

- Use scheduling tool (Buffer, Hootsuite)
- Create campaign brief (goals, messaging, hashtags)
- Batch-create all assets before launch
- Run paid promotion for key posts (boost reach)
