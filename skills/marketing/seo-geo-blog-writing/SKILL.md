# SEO Geo Blog Writing

## Quick Start

This skill helps you write SEO-optimized content for specific locations. For complete frameworks, see the [Complete Guide](references/guide.md).

## Execution Checklist

### Writing Geo-Targeted Content

- [ ] Research location-specific keywords
- [ ] Analyze local search intent
- [ ] Include location in title, H1, URL
- [ ] Add local context (landmarks, businesses, events)
- [ ] Include geo-specific statistics
- [ ] Add local schema markup
- [ ] Link to local resources
- [ ] Create location-specific CTAs
- [ ] Track local rankings

## Key Frameworks

- **Geo-Keyword Formula**: `[Keyword] + [City/Region]`
- **Local Context**: References to landmarks, local businesses, events
- **Location Signals**: Address, phone, reviews, local backlinks
- **Multi-Location**: One page per location (avoid duplicate content)

## Quick Wins

- Add city name to title tag naturally
- Reference local landmarks in content
- Use local testimonials/case studies
- Create location-specific landing pages
