# User Onboarding

## Quick Start

This skill helps you optimize user onboarding. For complete frameworks, see the [Complete Guide](references/guide.md).

## Execution Checklist

### Optimizing Onboarding

- [ ] Define "aha moment" (activation event)
- [ ] Map user journey to value
- [ ] Remove friction (reduce steps)
- [ ] Add progress indicators
- [ ] Use tooltips/walkthroughs
- [ ] Track activation rate
- [ ] A/B test onboarding flows
- [ ] Measure time-to-value

## Key Frameworks

- **Aha Moment**: First experience of core value
- **Time-to-Value**: Minutes/hours to activation
- **Onboarding Checklist**: Visual progress bar
- **Empty States**: Guide next actions

## Activation Metrics

- **Activation Rate**: % who reach aha moment
- **Time-to-Activation**: Days to first value
- **Target**: D7 activation >40%, D30 >60%

## Quick Wins

- Single-step signup (defer details)
- Empty state guidance ("Add your first X")
- Progress checklist (5 steps to success)
- Quick win in first session
