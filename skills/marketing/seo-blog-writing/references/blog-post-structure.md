# SEO Blog Post Structure Framework

## Overview
A proven structure for blog posts that rank on Google and convert readers into subscribers.

## Complete Structure

### 1. Title (H1)
**Goal**: Rank for target keyword + entice clicks

**Formula**:
```
[Number] [Adjective] Ways to [Achieve Result] [Qualifier]

Examples:
- "7 Proven Ways to Increase Email Open Rates in 2024"
- "The Complete Guide to Cold Email (With Templates)"
- "How to Write Landing Page Copy That Converts [Step-by-Step]"
```

**Best Practices**:
- Include target keyword
- 50-60 characters (for SERP display)
- Use power words (proven, complete, ultimate)
- Add year or qualifier for freshness

### 2. Introduction (150-200 words)
**Goal**: Hook reader and promise value

**Structure**:
```
Paragraph 1: Hook (problem or bold claim)
Paragraph 2: Why this matters (stakes)
Paragraph 3: What they'll learn (preview)
```

**Example**:
```
95% of blog posts get zero traffic from Google. (Hook)

That's because most people write for readers, not search engines.
The result? Great content that nobody finds. (Stakes)

In this guide, you'll learn the exact SEO framework I use to rank
#1 for competitive keywords. I'll show you keyword research,
on-page optimization, and link building tactics that actually work. (Preview)
```

### 3. Table of Contents
**Goal**: Improve UX and create jump links

**Format**:
```
## Table of Contents
1. [Section 1 Title](#section-1)
2. [Section 2 Title](#section-2)
3. [Section 3 Title](#section-3)
```

**Benefits**:
- Helps readers scan
- Creates featured snippet opportunity
- Improves dwell time

### 4. Main Content (H2 Sections)
**Goal**: Deliver value and target semantic keywords

**Structure per section**:
```
## H2: Section Title (Include Keyword Variation)

Intro paragraph (what this section covers)

### H3: Subsection 1
Content with examples

### H3: Subsection 2
Content with examples

[Visual: Screenshot, diagram, or chart]

Key takeaway or summary
```

**Best Practices**:
- Use H2 for main sections (4-7 sections)
- Use H3 for subsections
- Include keyword in first H2
- Add visuals every 300-500 words
- Use short paragraphs (2-3 sentences)

### 5. Examples/Case Studies
**Goal**: Prove concepts with real data

**Format**:
```
### Example: [Specific Case]

**Situation**: [Context]
**Action**: [What was done]
**Result**: [Specific outcome with numbers]

[Screenshot or visual proof]
```

### 6. Actionable Tips/Checklist
**Goal**: Make content immediately useful

**Format**:
```
## [Topic] Checklist

Before publishing your [thing]:

- [ ] Item 1 with specific action
- [ ] Item 2 with specific action
- [ ] Item 3 with specific action
```

### 7. FAQ Section
**Goal**: Target "People Also Ask" keywords

**Format**:
```
## Frequently Asked Questions

### Question 1 (from keyword research)?
Direct answer in first sentence.
Additional context and details.

### Question 2?
Direct answer in first sentence.
Additional context and details.
```

**Best Practices**:
- Use actual questions from Google
- Answer in first sentence
- Expand with 2-3 paragraphs

### 8. Conclusion
**Goal**: Summarize and drive action

**Structure**:
```
Paragraph 1: Recap main points
Paragraph 2: Next steps
Paragraph 3: CTA (comment, share, download)
```

**Example**:
```
We've covered the complete SEO blog framework: keyword research,
on-page optimization, and content structure.

Start by implementing the title and H2 optimization today.
Then move to internal linking and FAQ sections.

Want my full SEO checklist? Download it here: [Link]
```

### 9. Internal Links
**Goal**: Improve SEO and keep readers on site

**Strategy**:
- Link to 3-5 related posts
- Use descriptive anchor text
- Link early in the post
- Link to pillar content

**Example**:
```
"For more on email marketing, check out our 
[complete guide to email sequences](link)."
```

### 10. External Links
**Goal**: Build credibility and provide sources

**Best Practices**:
- Link to authoritative sources
- Use for statistics and claims
- Open in new tab
- 2-5 external links per post

## Keyword Optimization

### Primary Keyword Placement
- [ ] In title (H1)
- [ ] In first H2
- [ ] In first 100 words
- [ ] In URL slug
- [ ] In meta description
- [ ] In image alt text

### Keyword Density
- **Target**: 1-2% of total words
- **Natural usage**: Don't force it
- **Variations**: Use semantic keywords

### LSI Keywords
Include related terms:
- Synonyms
- Related concepts
- Question variations

## Meta Elements

### Title Tag
```
[Primary Keyword] | [Brand Name]
Example: "Email Marketing Guide | MarketingPro"
```
- 50-60 characters
- Include primary keyword
- Add brand name

### Meta Description
```
Learn [primary keyword] with our [adjective] guide. 
Includes [benefit 1], [benefit 2], and [benefit 3]. 
[CTA].
```
- 150-160 characters
- Include primary keyword
- Add CTA
- Write for clicks, not just SEO

### URL Slug
```
/primary-keyword-secondary-keyword
Example: /email-marketing-guide
```
- Short and descriptive
- Include primary keyword
- Use hyphens, not underscores
- Lowercase only

## Content Length Guidelines

**Target Word Count by Intent**:
- **Informational**: 1,500-2,500 words
- **How-to/Tutorial**: 2,000-3,000 words
- **Ultimate Guide**: 3,000-5,000 words
- **Listicle**: 1,000-2,000 words

## Visual Elements

**Required**:
- Featured image (1200x630px)
- Screenshots (every 300-500 words)
- Diagrams for processes
- Charts for data

**Optional**:
- Videos (embedded)
- Infographics
- GIFs for demonstrations

## Success Metrics
- **Organic traffic**: 500+ visits/month (after 3-6 months)
- **Average position**: Top 10 for primary keyword
- **Dwell time**: 3+ minutes
- **Conversion rate**: 2-5% to email list
