# SEO Workflow Orchestration

## Quick Start

This skill helps you build automated SEO workflows. For complete frameworks, see the [Complete Guide](references/guide.md).

## Execution Checklist

### Building SEO Workflow

- [ ] Set up keyword research process
- [ ] Create content brief template
- [ ] Build editorial calendar
- [ ] Assign content to writers
- [ ] Implement SEO checklist for editors
- [ ] Set up publishing workflow
- [ ] Automate rank tracking
- [ ] Schedule content audits (quarterly)
- [ ] Create reporting dashboard

## Key Components

- **Keyword Research**: Tools + Process + Tracking
- **Content Briefs**: Template with keyword, intent, outline, examples
- **Production**: Assignment → Draft → Edit → SEO Review → Publish
- **Optimization**: Track rankings → Update underperformers → Build backlinks

## Quick Wins

- Use Airtable or Notion for editorial calendar
- Create content brief template (saves 30 min per post)
- Set up automated rank tracking (weekly)
- Schedule content refresh every 6 months
