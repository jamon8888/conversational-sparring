# Brand Guidelines

## Quick Start

This skill helps you create brand guidelines. For complete frameworks, see the [Complete Guide](references/guide.md).

## Execution Checklist

### Creating Brand Guidelines

- [ ] Define brand mission and values
- [ ] Create logo usage rules (sizes, spacing, don'ts)
- [ ] Set color palette (primary, secondary, accent colors with hex codes)
- [ ] Choose typography (headings, body, web fonts)
- [ ] Define voice and tone (personality traits)
- [ ] Create content guidelines (writing style, terminology)
- [ ] Design visual examples
- [ ] Share with team

## Key Components

- **Logo**: Primary/secondary versions, minimum sizes, clear space
- **Colors**: Primary (3), Secondary (3), with HEX/RGB/CMYK
- **Typography**: 2-3 fonts max (heading, body, accent)
- **Voice**: 3-5 personality traits (friendly, professional, witty)
- **Imagery**: Photo style, illustration style, dos/don'ts

## Quick Wins

- Use Canva or Figma templates
- Create one-page quick reference
- Share as PDF + living Google Doc
- Include real examples (good vs. bad)
