# Engineering Performance Metrics

## 1. DORA Metrics (Delivery Performance)

- **Deployment Frequency**: Target >1/day.
- **Lead Time for Changes**: Target <1 day from commit to prod.
- **Mean Time to Recovery (MTTR)**: Target <1 hour.
- **Change Failure Rate**: Target <15%.

## 2. Quality KPIs

- **Test Coverage**: Target >80% for critical paths.
- **Code Review Rate**: Target 100% of pull requests reviewed.
- **Technical Debt Ratio**: Target <10% of code base or capacity.
- **Escaped Bugs**: Target reducing trend per release.

- **Manager to Engineer**: 1:8
- **Senior to Mid to Junior**: 3:4:2
- **Product to Engineering**: 1:10
