# Product Metrics & Analytics

## 1. North Star Metric (NSM)

The single metric that best captures the core value your product delivers:

- **Measurable**: Quantitative and trackable.
- **Actionable**: The team can directly influence it.
- **Leading Indicator**: Predicts long-term business success (Revenue/LTV).

## 2. Pirate Metrics (AARRR)

- **Acquisition**: Where users come from.
- **Activation**: The "Aha!" moment.
- **Retention**: Users coming back.
- **Revenue**: Monetization events.
- **Referral**: Users inviting others.

## 3. Feature Success Framework

- **Adoption**: % of total user base using the feature.
- **Frequency**: How often the feature is used per time period.
- **Depth**: How much of the feature's capability is explored.
- **CSAT/NPS**: Qualitative satisfaction with the specific feature.

## 4. Key Metrics Rules

- Avoid **Vanity Metrics** (e.g., total registered users).
- Focus on **Actionable Metrics** (e.g., conversion rate from free to paid).
- Use **Cohorts** to track how changes affect specific groups over time.
