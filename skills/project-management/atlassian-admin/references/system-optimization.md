# System Performance & Integrations

## 1. Performance Optimization

### Jira Excellence

- **Archiving**: Periodically archive old Jira projects and issues.
- **Reindexing**: Schedule during off-peak hours to restore search performance.
- **Scheme Cleanup**: Remove unused workflows, fields, and notification schemes.

### Confluence Excellence

- **Archiving**: Move inactive spaces to an archive state.
- **Storage**: Remove large orphaned attachments and compress image data.
- **Cache**: Monitor system health for macro-driven performance lag.

## 2. Integration Catalog

- **Slack/Teams**: Real-time notifications for issue transitions.
- **GitHub/Bitbucket**: Link code commits and PRs to Jira tickets.
- **Salesforce**: Support ticket sync for customer success visibility.
- **Miro/Figma**: Embedding dynamic design boards into Confluence pages.

## 3. Marketplace Governance

- **Sandboxing**: Always test new apps in a sandbox environment before production.
- **Security Check**: Verify vendor security certifications (e.g., Cloud Fortified).
- **Audit**: Review app usage annually to reduce license waste.
